# chat_panel_frame

Create a tall reusable right-side chat panel on a plain clean background, with parchment tones, brown pixel outlines, header area for agent avatar and name, status row, stacked message bubble areas, a blue player bubble style, a lighter agent bubble style, a bottom input bar, and a send button. It should feel like part of the same cozy office game UI system, suitable for frontend implementation reference. Crisp pixel art, no modern SaaS styling.
