# workstation_state_sheet

Show four workstation states arranged in a clean horizontal sprite-sheet style on a neutral plain background with spacing between them: 1) empty workstation base, 2) hovered workstation with soft golden pixel outline, 3) selected workstation with stronger golden outline and subtle floor highlight plate, 4) occupied workstation desk base ready for a seated worker. Each workstation should include a honey wood desk, dark office chair, monitor, keyboard, and small desk props. Crisp pixel art, strong outlines, cozy office colors, no UI text.
