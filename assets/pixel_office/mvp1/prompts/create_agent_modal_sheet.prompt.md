# create_agent_modal_sheet

Show two pixel-art UI panels on a plain clean background: 1) a centered create-agent confirmation modal with parchment tones, brown pixel outline, green primary button, orange secondary button, small pixel close button; 2) a create-agent form modal with title area, input field styling, dropdown styling, avatar preview box, and create button. The look should feel like in-world game UI, not SaaS web UI. Crisp pixel art, warm parchment colors, reusable UI skin style, no extra decorative scene background.
