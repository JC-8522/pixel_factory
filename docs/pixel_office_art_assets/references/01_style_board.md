# Style Board

## Style Summary

The target look is:

- Stardew-inspired pixel office
- cozy and handcrafted
- readable from a management-game zoom level
- playful but operational

## Core Style Anchors

- deep wooden frame around the room
- warm cream wall
- blue-gray tile floor
- chunky dark outlines
- light honey-colored desks
- pixel furniture around room edges
- dark office chairs
- floating plaque labels for desks
- seated workers integrated into desks
- parchment-like in-world modal and chat UI

## Avoid

- generic dashboard cards
- modern blur or glass effects
- pure dark mode admin styling
- soft vector icons
- oversized smooth rounded corners
- empty black or flat-color backgrounds

## Target Feeling

The scene should feel like:

- a tiny game office the user manages,
- a place that remains attractive even before agents are created,
- an office floor that can visually scale from empty to full occupancy.
