# Prompt 02: Workstation

## Purpose

Generate the workstation art direction and state variants.

## Prompt

Create a pixel-art office workstation in a Stardew Valley-inspired style. The workstation should include a light wood desk, a dark office chair, dual monitors, a keyboard, and one or two small desk props like a coffee mug, sticky notes, or a small plant. Produce the workstation so it can support four states: empty, hovered, selected, and occupied. The empty state should look clean and ready to use. The hovered state should feel highlighted like a selectable game tile with a soft golden outline. The selected state should feel more active and intentional, with a stronger golden border and a subtle floor highlight plate. The occupied state should feel like a real working desk that can seat a small pixel worker. Preserve crisp pixel art, dark outlines, and warm cozy office colors.

## Output Intent

- workstation base
- hover overlay
- selected overlay
- occupied workstation composition reference
