# Prompt 03: Agent Sprite

## Purpose

Generate the first pixel worker direction for seated office agents.

## Prompt

Create a seated pixel-art office worker in a Stardew Valley-inspired style. The character should look like a charming small game NPC sitting at a workstation and facing the monitor, with readable hair, face direction, torso, arms, and legs. The worker should fit naturally into a cozy office desk setup and look believable as part of a management sim scene. Create state variations for idle, working, thinking, and blocked. Use expressive but compact pixel shapes, clear silhouette, warm outlines, and outfit colors that work well inside a wood-and-blue office scene. Avoid anime exaggeration, hyper-realistic rendering, or modern vector UI style.

## Output Intent

- base seated worker design
- state variation guide
- avatar direction for multiple future agents
