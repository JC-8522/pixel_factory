# Prompt 04: Chat And Modal UI

## Purpose

Generate the create-agent modal and right-side chat drawer visual skin.

## Prompt

Create a set of pixel-art UI panels in a Stardew Valley-inspired office game style. Include a centered create-agent confirmation modal, a create-agent form modal, and a right-side agent chat drawer. The panels should use warm parchment or cream paper tones, dark brown pixel outlines, subtle inner borders, and game-like button styling. The primary button should be green and friendly. The secondary button should be orange. The chat drawer should feel like part of the same game world, with a header area for a pixel avatar, a status row, parchment-like message areas, a blue player message bubble, a lighter agent response bubble, and a pixel input bar with a send button. Avoid modern SaaS glassmorphism or rounded mobile-app styling.

## Output Intent

- modal frame direction
- chat panel frame direction
- reusable button and input skin direction
